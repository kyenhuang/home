前言

本文介绍如何使用Plantuml进行有关设计。
编写本文的目的是记录个人学习PlantUML的过程，一部知识来源于官网。
以及在学习过程中遇到问题和解决方案。

什么是PlantUML？

PlantUML是一个快速创建UML图形的组件。
PlantUML支持的图像有：

  • Squence Diagram
  • Class Diagram
  • Use Case Diagram
  • Activity Diagram
  • Component Diagram
  • State Diagram
  • Object Diagram
